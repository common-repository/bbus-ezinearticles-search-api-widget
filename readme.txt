== Description ==

BBU’s EzineArticles Search API Widget is a plugin that allows you to search EzineArticles.com repository of articles using the standard API calls (not scraping) and displays those results in WordPress widget.

EzineArticles lets you performs 100 requests per hour. This plugin is so efficient you only need no more than one request per hour, leaving you many remaining requests for other applications.

Go to BBU's [EzineArticles Search API Widget](http://blogbuildingu.com/software/ezinearticles-search-api-widget "BBU's EzineArticles Search API Widget") plugin page for complete information.

== Installation ==

Installing should take no longer than two minutes.

1. Upload the directory eza-searchapi to your /wp-content/plugins directory.
2. Go to the plugin management page and activate the plugin.
3. That’s it!

== Frequently Asked Questions ==

= Will the content in the widget update automatically? =

Yes, once you set the options up, this plugin will regularly fetch the updated content for you so it displays the most up to date data all the time.
